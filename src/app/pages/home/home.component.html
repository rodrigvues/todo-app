O Todo é um web app desenvolvido por Vitor Rodrigues para estudar linguagens como .NET e Angular, além de banco de dados PostGres. Um simples CRUD de tarefas com foco no front end.

O foco é aprender as linguagens e a conexão delas, usando Swagger, Prime NG e pgAdmin no processo de aprendizado.
